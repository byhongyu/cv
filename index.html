Side Projects

  • A MapReduce Application for Company Recommendation

    * Proposed a connectivity-based recommendation system. The weight values are calculated from connections obtained in a social network graph that captures commercial partnerships. For each Mapper, instead of the intuitive consideration of the DFS, it will assume that each node is the 2-hop node of its college. For each Reducer, it stores all the pairs in the HashMap based on the key, then sorts it based on the company as well as connectivity.
    * The method used in this problem can be extended to the areas like the friend recommendation in social media and the movie recommender in video websites.
    Technologies used: MapReduce, Eclipse, java

  • A New Parallelization Method for K-means 

    *  Proposed an improved parallel method for k-means, IPKMeans, which has a parallel preprocessing stage using k-d tree and can finish k-means in one single MapReduce job with much more reducers working in parallel and has a fast post-processing stage generating the final result. In our method, both k-d tree and the new improved parallel k-means are implemented using MapReduce and tested on Hadoop.
    * Research results were summarized into a manuscript (see Publication [1])
    Technologies used:  Hadoop, MapReduce, Eclipse, java.
